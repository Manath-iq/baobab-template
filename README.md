# Баобаб

Одностраничный сайт семейной мастерской шоколада «Баобаб» в Нижнекамске.

## Локальный просмотр

```sh
python3 -m http.server 4173
```

Откройте `http://127.0.0.1:4173`.

## Публикация

Сайт автоматически публикуется в GitHub Pages после каждого пуша в ветку `main`.
Workflow находится в `.github/workflows/pages.yml`.
